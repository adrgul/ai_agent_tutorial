# Python AI Agent System

This project scaffolds a production-grade Python project that uses LangGraph for agent orchestration and Groq for LLM inference, strictly adhering to Clean Architecture principles.

## Setup

1.  **Install `uv`**:
    ```bash
    pip install uv
    ```

2.  **Install dependencies**:
    ```bash
    uv pip install -e .
    ```

3.  **Environment Variables**: Create a `.env` file based on `.env.example` and fill in your API keys.

4.  **Run the application**:
    ```bash
    python main.py
    ```
